Regex
	python regexes
	. - any character except new line - to match new line read re.DOTALL
	\ - escape char, escape any wildcard
	^ - beggining of the line
	$ - end of the line
	* - match the preivous char, from 0 to many times
	+ - match the preivous char, from 1 to many times
	? - match the previous char, 0 or 1
	{n} - match the previous char, n times
	{n, m} - match the previous char, from n to m times
	[abc] - match either a, b or c
	[a-z] - match char from a to z
		you can use [a-c], [b-g], etc
	[a-z0-9] - match any char or number
	a|b - match a or b - use parenthesis to speific what
	() - capture groups
	^a - match anything excepts a
	\w - match any alphanumeric (the same as [aa-zA-Z0-9_] incluindo não ascii)
	\d - match any numeric (the same as [0-9])
	\D - match any non numeric char (the same as [^0-9])
	\s - any whitespace (acho que tab tb)
	\S - any non whitespace
	metacharacter não são ativos dentro de []
